3205. The name "Boolean" conflicts with Windows headers

Section: 18.5.2 [concept.boolean] Status: New Submitter: Casey Carter Opened: 2019-05-10 Last modified: 2019-05-11

Priority: Not Prioritized

View all issues with New status.

Discussion:

The Boolean concept was recently added to the C++20 working draft. Unfortunately, there are Windows platform headers that use the name "Boolean" in the global namespace. Programs that import namespace std into the global namespace via using directive — as is common in the Windows codebase — will be broken by the C++20 Standard Library if this name is not changed.
