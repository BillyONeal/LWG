3205. The name "Boolean" is commonly macroized

Section: 18.5.2 [concept.boolean] Status: New Submitter: Casey Carter Opened: 2019-05-10 Last modified: 2019-05-12

Priority: Not Prioritized

View all issues with New status.

Discussion:

The Boolean concept was recently added to the C++20 working draft. Unfortunately, the name "Boolean" is commonly defined as a macro in the Windows codebase. There are a few dozen occurrences of "#define Boolean" in ACTCD19, so this issue exists to a lesser extent in other code as well. Using "Boolean" as the name of the concept will inhibit adoption in code that uses such a macro definition.
